## Notes

### Breaking Changes in 2.0.0
- File paths changed due to reorganization
- Color scheme completely changed
- Font families updated
- Some class names modified for consistency

### Migration Guide (1.0.0 ‚Üí 2.0.0)
If you have customizations from version 1.0.0:

1. **Update file paths**:
   - `styles.css` ‚Üí `css/styles.css`
   - `script.js` ‚Üí `js/script.js`

2. **Update colors**:
   - Old: `#ff6b6b`, `#f06595`
   - New: `#1e3a8a`, `#3b82f6`

3. **Update fonts**:
   - Old: Outfit, Poppins
   - New: Montserrat, Roboto

4. **Check custom CSS**:
   - Review any custom styles
   - Update to match new design system
